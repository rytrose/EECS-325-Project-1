This proxy uses port 50067, as my class list number is 67.

To use the proxy, compile 'javac proxyd.java', then run 'java proxyd - port 50067'.
Then, point your browser at the running proxy and send HTTP requests via your browser.

I tested this proxy using case.edu, 4chan.org, cnn.com, and imgur.com.
I used Google Chrome as my browser.
